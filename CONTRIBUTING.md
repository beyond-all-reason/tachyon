# Contributing to Beyond All Reason (BAR)

## Building the project

Look at the commands under [the ci action](./.github/workflows/ci.yml) to see what's required.
You should only ever need to manually touch files under `src/`. Everything under `schema/` and
`docs` is autogenerated.


## AI Policy

Refer to the [AI Usage Policy](AI_POLICY.md) if you used an AI to generate production code.
